## nodeGPS

# Description
A academic project in order to optimize GPS data compression and sending system.

# Modules used

- ESP8266;
- NEO-7M GPS SENSOR;

# Libraries
- TinyGPS++;
- SoftwareSerial (modified);
- ArduinoJson;
- ESP8266WiFi;
- Huffman Compression;
